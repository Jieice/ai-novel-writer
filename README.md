## 🛠️ 核心技能系统

### 必装技能（安装到 Trae）

| 技能 | 触发词 | 功能 |
| --- | --- | --- |
| **novel-project-starter** | 写新小说、开新书 | 创建项目、生成大纲 |
| **novel-project-sync** | 同步、写完了、降重 | 降重+同步项目文件 |
| **novel-publish-fanqie** | 发布到番茄 | 导出txt到发布目录 |
| **novel-style-learner** | 学习风格、分析参考书 | 分析参考小说风格 |
| **novel-conflict-designer** | 毒点、冲突设计 | 检测毒点、设计冲突 |
| **novel-master-planner** | 全局规划、创作进度 | 整体规划、进度追踪 |
| **novel-market-analyzer** | 市场分析、热门题材 | 题材分析、收益预估 |

## 📋 使用指南

### 1. 新建项目
```
说："我想写一本90年代乡土小说"
→ 自动创建项目、生成大纲和人物设定
```

### 2. 学习风格（可选）
```
说："学习《乡村那些事儿》的风格"
→ 分析参考书、生成风格指南
```

### 3. 写作章节
```
说："写第1章"
→ 按大纲和风格指南写作
```

### 4. 同步降重
```
说："同步" 或 "写完了"
→ 自动降重、更新项目文件
```

### 5. 发布番茄
```
说："发布到番茄"
→ 导出txt到发布目录
→ 运行 python publish.py 自动上传
```

## 🔧 工具安装

### 1. 安装技能到 Trae

```
复制技能文件夹到：
C:\Users\{用户名}\.agents\skills\

需要安装的技能：
- novel-project-starter
- novel-project-sync
- novel-publish-fanqie
- novel-style-learner
- novel-conflict-designer
- novel-master-planner
- novel-market-analyzer
```

### 2. 安装番茄发布工具

```bash
cd fanqie_auto_publish
pip install playwright pillow
playwright install chromium
python login.py  # 扫码登录
```
