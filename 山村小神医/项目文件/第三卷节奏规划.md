## 详细章节规划

### CH31 贵人身份揭晓（新起点）

**节奏**: 新阶段开启，铺垫为主

**核心事件**:
- 镇长赵德明带"贵人"来访——省城药材公司经理周济民（周济生侄子）
- 周济民表示愿意合作，邀请陈大山去省城考察
- 林秀梅主动提出一起去（感情线推进）
- 刘瘸子私下提醒："省城水深，你爷爷的事可能跟那边有关"

**爽点**:
- 省城大公司主动合作
- 林秀梅愿意同行（感情升温）
- 镇长亲自引荐

**钩子设计**:
- 章中钩1: 周济民提到"省城有人专门收百年野山参"
- 章中钩2: 刘瘸子暗示爷爷往事
- 章中钩3: 林秀梅说"我陪你"
- 章中钩4: 周济民说"到了省城，有些事你自己判断"
- 章末大钩: 出发前夜，收到一封匿名信："别去省城，有人要害你"

**期待感管理**:
- 大期待：省城之行能否顺利？野山参能否到手？
- 小期待：匿名信是谁写的？

**必须推进伏笔**:
- F-018（百年野山参）→ 确认省城有线索
- F-019（贵人要来）→ ✅ 揭示是周济民
- F-017（爷爷灵珠）→ 暗示与省城有关

---

### CH32 省城初到（小高潮）

**节奏**: 上升通道，小爽点

**核心事件**:
- 陈大山和林秀梅抵达省城，周济民安排住处
- 周济民带陈大山见"野山参主人"——一位退休老中医
- 老中医愿意转让百年野山参，但要陈大山帮他治一个疑难杂症
- 陈大山用灵珠诊断，发现老中医体内有慢性毒素

**爽点**:
- 省城见世面
- 野山参有门路了
- 医术得到认可

**钩子设计**:
- 章中钩1: 省城繁华景象，陈大山第一次进城
- 章中钩2: 老中医说"你爷爷我认识"
- 章中钩3: 老中医体内毒素来源可疑
- 章末大钩: 老中医说"治好我，野山参给你。但你要小心，有人不希望你拿到它"

**期待感管理**:
- 大期待：能否治好老中医？野山参能否到手？
- 小期待：老中医认识爷爷？毒素是谁下的？

**必须推进伏笔**:
- F-018（百年野山参）→ 确认位置，有条件
- F-001（爷爷手札遭祸）→ 老中医认识爷爷

---

### CH33 野山参被抢（挫折）

**节奏**: 全章压抑，最大挫折

**核心事件**:
- 陈大山准备治疗老中医，但当天夜里老中医家被盗
- 百年野山参被偷走，老中医急得旧病复发
- 陈大山调查发现，是李万山的残党派人干的
- 林秀梅被跟踪，陈大山意识到对方在威胁他

**挫折感**:
- 野山参近在咫尺却丢了
- 对手在暗处，自己在明处
- 林秀梅被牵连，愧疚

**钩子设计**:
- 章中钩1: 老中医家一片狼藉
- 章中钩2: 监控录像里的人眼熟
- 章中钩3: 林秀梅说"有人在跟踪我"
- 章中钩4: 陈大山发现灵珠在微微发热
- 章末大钩: 灵珠突然指向一个方向——野山参还在省城

**期待感管理**:
- 大期待：能否找回野山参？
- 小期待：灵珠为什么发热？新能力？

**必须埋伏笔**:
- 🌱 F-020: 灵珠有追踪能力（二级预兆）

### CH34 灵珠异动（转折）

**节奏**: 上升通道，意外之喜

**核心事件**:
- 陈大山顺着灵珠指引，在省城药材市场找到一株珍稀药材（第九种）

- 陈大山追踪到野山参位置——省城某药材仓库
- 周济民派人帮忙，准备明天行动

**爽点**:
- 意外收获第九种药材
- 灵珠新能力觉醒
- 找到野山参下落

**钩子设计**:
- 章中钩1: 灵珠指引发现珍稀药材
- 章中钩2: 灵珠升级预兆——开始发热
- 章中钩3: 追踪到仓库位置
- 章末大钩: 仓库门口有人把守，陈大山发现其中一人是——张二狗

**期待感管理**:
- 大期待：明天能否夺回野山参？
- 小期待：张二狗怎么在省城？李万山残党？

**必须推进伏笔**:

## 读者留存设计

### 开篇200字抓人模板（CH31）
```
[直接切入冲突/悬念] + [主角立即行动] + [新信息点]
示例：镇长带着一个陌生人来了，说是省城的大老板。陈大山还没反应过来，那人就开口了："你爷爷的灵珠，我知道在哪。"
```

### 章中钩子模板（每章3-4个）
```
位置：每800-1200字一个
类型：
1. 微型反转：情况突变
2. 信息差：读者知道角色不知道
3. 新疑问：抛出问题
4. 情绪波动：角色反应异常
```

### 章尾悬念模板
```
类型：
1. 纵向钩（悬念钩）：揭晓前一秒戛然而止
2. 横向钩（爽点钩）：爽点完成→立刻抛出新危机
3. 人物钩（关系钩）：用情绪张力代替外部事件
```

### 情绪闭环设计
```
每章结构：压抑→反转→爽感→新期待
示例：野山参被抢（压抑）→灵珠追踪（反转）→找到位置（爽感）→张二狗出现（新期待）
```
