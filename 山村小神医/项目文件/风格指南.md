# 写作风格指南：人味

> 朱雀实测结论：第三人称AI无法洗掉自己的统计指纹。任何由AI生成的文本（第三人称），无论怎么精修/重写/换词/打断句长，朱雀都能识别（AIGC=0.9984）。
> **但是！天眼AI实测验证：第一人称+8条铁律写法，AI率明显降低！**
> 本指南的目标不是"骗过AI检测器"，而是**从根本上改变写作方式**——让文本从生成逻辑上就不像AI。
> 
> **核心参照**：用户本人写的小说片段（见附录），这是"人味"的真实样本。
> **检测平台**：天眼AI（tianyanai.org），完全免费

## 一、人写东西 vs AI写东西：本质区别

| 维度 | AI写的东西 | 人写的东西（你的样本） |
|------|-----------|----------------------|
| 逻辑 | 完整、闭环、前后照应 | 松散、跑题、想到哪说哪 |
| 情绪 | 准确命名（"愤怒""悲伤"） | 不说出来，用笨拙的方式表达 |
| 过渡 | 每段都有承接 | 硬切，或者用"因为""而"这种不讲究的连词 |
| 信息密度 | 每句都有功能 | 有些话就是废话，有些细节就是顺手的 |
| 完整度 | 句句完整、语法正确 | 句子可以没说完，可以语病 |
| 自我暴露 | 主角永远体面 | 主角会承认丢人的事 |
| 幽默 | 设计好的包袱 | 朋友间的烂笑话、只有自己人懂的梗 |
| 节奏 | 均匀推进 | 一会儿聊天聊很久，一会儿跳好几天 |
| 结尾 | 钩子/悬念/收束 | "时间又过去了几天"——就这么算了 |

## 二、从真人写作中提取的8条铁律

### 铁律1：允许自己丢人

你的样本里最"人"的地方就是自我暴露：

> "我只牵过女孩子的手，连嘴都没碰过，我只是不想一个人偷偷在深夜的被子里流泪罢了"

AI永远不会让主角说这种话。AI的主角永远体面、克制、有尊严。但真人写东西会把自己最丢人的一面掏出来。

**写法**：每章至少1处主角的"丢人时刻"——说错话、做蠢事、暴露弱点。不美化。

### 铁律2：废话就是人味

你的样本里大量"没用的信息"：

> "他和我一样，不太习惯来公共场所吃饭，也显得拘谨了一些"——这个信息对剧情没用，但让人觉得真实
> "美国肥牛我属实没印象了，我只记得他们说是有个小零食叫这个名字"——完全不推动剧情，但这就是人说话的方式

AI的每句话都在"推进剧情"或"塑造人物"，太有目的性了。人写东西会走神。

**写法**：每5-8段加1处"没功能"的信息——回忆、跑题、随手细节。

### 铁律3：朋友对话是废话连篇

你的对话是最像人的部分：

> "你个臭XX别玩了，我看是菜快上了"
> "都怪你，死了啦"
> "我要吃美国肥牛！"

这些对话——没信息量、没文学价值、甚至没礼貌。但这就是朋友说话的样子。AI写的对话太"有水平"了，每句都有潜台词、每个字都在塑造角色。人的对话就是乱七八糟的。

**写法**：朋友间的对话至少1/3是废话/玩笑/起哄。不要每句都有意义。

### 铁律4：情绪用笨办法表达

> "我也没关"（应该是"也没关系"，但你写的是口语化的缩略）
> "因为我不知道怎么去处理......"（句末省略号，人写东西才会这样——不确定、尴尬）

AI写情绪是"眼眶一热""心里一紧"，太文艺了。你的写法是直接承认"我害怕""我不知道怎么处理"——笨，但真。

**写法**：情绪不用比喻/意象，直接说"不知道""害怕""尴尬"。用省略号表示说不下去。

### 铁律5：不讲究的过渡

你的过渡词极其朴素：

> "因为开学季快来了"——用"因为"开头，小学生都会
> "而坐在我右手边的是"——"而"字开头，最简单的转折
> "等菜上齐后"——最朴素的时间过渡
> "时间又过去了几天"——最敷衍的时间跳转

AI的过渡太讲究了——"暮色四合""日影西斜""话分两头"。人写东西就是"过了几天""因为""然后"。

**写法**：过渡词只用最朴素的——因为/所以/然后/过了/等。拒绝文学腔过渡。

### 铁律6：人物出场不用"设计"

> "坐在我对面的辰昊撇着脸"
> "在白杰左手边的是杨三水"
> "而坐在我右手边的是辰肖俊，他不说话，因为已经开了一把王者荣耀"

没有外貌描写、没有性格铺垫、没有"标志性动作"。就是告诉你在哪坐着、在干嘛。AI会让每个人物出场都"有设计感"——穿着、表情、小动作、身份暗示。人就是直接说"他在玩手机"。

**写法**：人物出场用位置+动作，不堆砌外貌和性格暗示。

### 铁律7：节奏说停就停，说跳就跳

你的样本前半段是吃饭聊天（很慢很细），然后：

> "在一顿'酒足饭饱'过后，我们各自散去"

一句话就把饭局结了。然后场景跳到报名处。中间没有任何"回到家里，躺在床上想了很多"之类的过渡。

AI会在场景之间做情感铺垫和逻辑过渡。人就是写完了就翻篇。

**写法**：慢的地方可以很慢（一顿饭写800字），快的地方一句话带过。不平均分配。

### 铁律8：内心独白是大白话

> "这次我绝对要在学校找个好看的"
> "我想快速离开这里"
> "真好吃"

AI的内心独白是"他暗暗下定决心""他心中暗想"。你的是直接说想要什么、想干什么，3个字搞定。

**写法**：内心独白不超过15个字，用口语，不用书面语。

## 三、反模式（必须避免）

| 反模式 | 表现 | 你的写法对照 |
|--------|------|-------------|
| 每段都有总结 | "他心里明白……" | 你从不总结，直接行动 |
| 情绪太准确 | "他感到一阵愤怒" | "我有点看不下去"——模糊但真 |
| 对话太有水平 | 每句都有潜台词 | "快吃啊，奶奶滴"——废话 |
| 过渡太丝滑 | "日影西斜，他回到了家" | "时间又过去了几天"——敷衍但真 |
| 人物出场太设计 | 外貌+性格+暗示 | "他在玩手机"——完了 |
| 句句完整 | 语法完美 | "我也没关"——口语缩略/语病 |
| 主角永远体面 | 永远克制、理性 | "连嘴都没碰过"——自我暴露 |
| 情绪用比喻 | "心里咯噔一下" | "我不知道怎么去处理"——笨但真 |

## 四、自检清单

改完一章后过一遍：

1. 🔍 主角有没有至少1处"丢人时刻"？
2. 🔍 有没有2处以上的"废话/跑题/没功能的细节"？
3. 🔍 朋友间对话有没有1/3是废话/起哄？
4. 🔍 情绪表达有没有用"笨办法"（直接说/省略号）而不是比喻？
5. 🔍 过渡词是不是只用了最朴素的（因为/然后/过了）？
6. 🔍 人物出场是不是位置+动作就完了？
7. 🔍 有没有"一句话带过"的快节奏段落？
8. 🔍 内心独白有没有超过15个字的？砍短。

"那你好好读书吧，别整天在家里看些低俗东西"坐在我对面的辰昊撇着脸，却装出一副成熟男人的样子。

而坐在我右手边的是辰肖俊，他不说话，因为已经开了一把王者荣耀，并且正在玩百里守约。

我怕显得拘谨，就望向了辰肖俊的手机，看他操作着，他打的确实有一套，可惜只喜欢玩无限乱斗。一边看着一边说:"什么飞屋啊，哈哈哈，没了！"

而他却没接话，而是认真的打着游戏，我也不语，继续望着窗外。

因为开学季快来了，大家有的不在一个学校，只能提前喊出来一起吃个饭。

"你个臭XX别玩了，我看是菜快上了。"杨三水说着，旁边的阿姨已经把菜端了上来。他也帮着整理桌子，好为下一次上菜提供空间。

"这次我绝对要在学校找个好看的，然后带她去岳池玩。"我说着，满眼都是菜，口中唾液蠢蠢欲动。

"你个菲比，还想着找啊，上一个人把你搞得只能读个普通高中了，你还想着放弃自己的学业啊"杨三水说道，并且拿起筷子吃了起来。

"快吃啊，奶奶滴，为什么不吃?"杨三水说道。随即辰昊和辰肖俊拿起筷子大口吃着。辰肖俊还抱怨到"都怪你，死了啦"

我不语，只是笑了笑，他们都是我初二认识的，到现在也得有三年交情了，我知道这是对我开玩笑，也没关，而是大口吃着菜。

"哎，我说这位小同志啊，别人的事就别管了，你说说你，你高中要去哪里读啊？"辰昊说到，学起老干部的模样细细的品了一口茶。他说得肯定是杨三水，因为初中连普通高中都没有考上，所以问了起来。
