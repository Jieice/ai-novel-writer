## 章节号防覆盖机制

**核心原则**：更新角色状态时，新章节号必须 ≥ 已记录的章节号，否则跳过更新

```
防覆盖规则：
- 心理状态：state_updated_chapter 新值 < 旧值 → 跳过
- 存活状态：status_changed_chapter 新值 < 旧值 → 跳过
```
