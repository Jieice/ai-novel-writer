# 山村小神医

**番茄小说推荐书名：《1992：回村后我成了神医》**

> 90年代+农村种田+神医金手指，番茄热门题材组合

## 故事简介

1992年，回村大学生陈大山意外发现爷爷留下的《百草手札》和神秘灵珠。

从此开启逆袭人生——

- 靠灵珠催动药材生长，从一介穷小子成为乡野神医
- 治好了被医院判了死刑的病人
- 驯服了全省唯一的百年野山参
- 更重要的是——他查清了爷爷当年死亡的真相

## 人物设定

**陈大山** - 主角
- 1992年省中医大专毕业回村
- 性格：隐忍型，先被动挨打再反击
- 金手指：百草灵珠（需收集10种珍稀药材升级）
- 目标：靠中医秘方发家致富，找出爷爷死亡真相

**林秀梅** - 女主角
- 22岁乡村代课老师
- 与主角暗生情愫，感情线贯穿全文

**赵老三** - 第一阶段反派
- 青山村一霸，与县城药材商勾结
- 处处刁难主角，最终被揭露真面目

**李万山** - 最终反派（隐藏boss）
- 县城药材公司副总
- 二十年前害死主角爷爷的真凶

## 金手指体系

**百草灵珠**（当前等级：1级，可升级）

| 能力 | 效果 | 限制 |
|------|------|------|
| 药材感知 | 50米内珍稀药材自动发光显示 | 无 |
| 加速生长 | 让药材3-5分钟内加速生长10% | 每日3次，用完需休息 |
| 品质提升 | 催动过的药材品质优于野生 | 过度使用会虚脱 |

**升级条件**：吸收10种珍稀药材（当前8/10，还差2种）
