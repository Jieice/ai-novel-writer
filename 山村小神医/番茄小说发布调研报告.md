### 1.1 番茄小说平台大盘

| 指标 | 数据 |
|------|------|
| 日活跃读者 | 1亿+（免费阅读赛道头部） |

| 2025年赛道趋势 | 从"拉新"转向"留存固化"，质量权重上升 |

### 1.2 神医流/乡村类赛道的真实情况

**结论先说：这个赛道有读者，但竞争确实激烈。**

- **有读者基础**：番茄上神医流、乡村种田文一直有稳定的读者群体。2024-2025年多篇推荐文章仍重点推荐这类题材，说明需求存在。
- **不是"红海中的蓝海"，但也绝非"死路"**：相比都市高武、无限流等番茄绝对头部类型，神医+乡村组合属于"老牌长青"品类，读者群稳定但增量不大。
- **你的差异化优势**：
  - ✅ **90年代回溯时间线**——目前大部分神医文是现代都市或架空年代，90年代乡镇背景是一个被忽视的时间窗口
  - ✅ **中医秘方+灵珠双金手指**——比纯"系统流"更有质感和代入感
  - ✅ **乡村成长→县城→省城的逐级升级**——节奏清晰，符合番茄"黄金三章"逻辑
- **隐忧**：
  - ⚠️ 神医流是"老品类"，读者有审美疲劳风险，前三章如果不能立住特色，很容易被划走
  - ⚠️ 番茄番茄小说2024巅峰榜TOP10中，神医/乡村类无一入选——说明这类作品在"爆火"层面有天花板

### 1.3 直接回答：真的会有人看吗？

**会有，但规模取决于前三章的留存和后续的完读率。**

番茄的推荐机制是"数据说话"：
1. 前几章的**完读率**决定是否进入下一级流量池
2. 完读率 > 20% 才有机会获得平台流量倾斜
3. 番茄2025年已进入"留存固化期"，留存质量 > 拉新数量

以你30章、约9.5万字的内容量，**足够在番茄启动第一波测试**。关键不在于"有没有人看"，而在于"看了能不能留下来"。

## 二、番茄发布的关键硬指标

### 2.1 签约门槛

| 项目 | 要求 |
|------|------|
| 最低字数 | 2万字可申请签约（你的9.5万已远超） |
| 连续更新 | 每日4000-6000字 |
| 全勤奖 | 月更≥10万字 + 不断更 + 听读收益≥500元 |
| 全勤金额 | 4000字/天 → 600-1500元/月；6000字/天 → 800-2000元/月 |

### 2.2 核心数据指标

| 指标 | 含义 | 你需要关注的阈值 |
|------|------|-----------------|
| **完读率** | 读者从头读到尾的比例 | ≥8%可参与星火计划；≥20%获流量倾斜 |
| **追读比** | 日更后当日阅读人数/总阅读人数 | 越高越好，影响推荐权重 |
| **在线时长** | 读者在你的书中停留总时长 | 平台核心考核指标 |
| **广告完读率** | 读者看广告的比例和完成率 | 直接关联分成收益 |

### 2.3 收入模型

番茄是免费阅读 + 广告分成模式，和起点/晋江的付费阅读完全不同：

```
月收入 ≈ 全勤奖(600-2000) + 听读分成(波动大) + 礼物分成(70%) + 版权分成(长期)
```

**新作者现实预期**：
- 前1-3个月：主要靠全勤奖，月入 600-2000 元
- 3个月后如果数据好：听读分成开始贡献，月入 2000-5000 元
- 头部作品：月入过万到数万（少数）
- **多数新作者的月收入在 1000-3000 元区间**

## 四、发布前必须优化的清单

按优先级排序，**🔴 必须做**的事项：

### 🔴 P0：必须解决（否则不建议发布）

#### 1. AI率全面降重（最关键）
- **现状**：多章AI率55-61%
- **番茄现状**：发布时必须勾选"是否使用AI"，平台有事后检测能力
- **不勾选的风险**：被检出后面临重罚、下架、封号
- **勾选的影响**：可能进入更严格审核队列，流量池受限
- **建议**：
  - 将所有章节AI率降至45%以下（目标40%）
  - 重点打磨对话：加入更多方言口语、90年代用语、不完整句
  - 增加生活细节描写：赶集、做饭、干农活等场景的感官细节
  - 参考你的参考小说库中的口语风格

#### 2. 前3章（黄金三章）重打磨
- **番茄铁律**：前三章决定完读率，完读率决定推荐
- **当前问题**：
  - 第1章仅2110字，偏短，爽感评分仅5/10
  - 第2-3章节奏偏慢，在"铺垫"而非"吸引"
- **建议打磨方向**：
  - 第1章前500字必须有**强情绪钩子**（被嘲笑/被羞辱/面临绝境）
  - 第1章结尾必须有**悬念钩子**（发现手札的瞬间要更震撼）
  - 第2-3章必须让读者看到**第一个"爽点"**（治好父亲的快感要放大）
  - 建议前三章总字数扩充到8000-10000字

#### 3. 书名和简介优化
- **当前书名**："山村小神医"——过于直白，在番茄搜索中会淹没在同类书名里
- **建议**：
  - 考虑加入90年代时间线或差异化关键词
  - 备选方案示例：
    - "回村1992：我靠祖传手札逆天改命"
    - "九二穷小子，开局觉醒百草灵珠"
    - "从村里到省城：一个中医大专生的逆袭"
  - 简介必须在3句话内说清：**主角是谁 + 金手指是什么 + 核心爽点是什么**

### 🟡 P1：强烈建议

#### 4. 章节字数统一
- 番茄读者习惯每章2000-3000字
- CH1(2110)、CH23(2036)、CH30(2184) 等偏短章节建议扩充
- CH19(4531)、CH17(4103) 等偏长章节建议拆分或精简

#### 5. 更新节奏规划
- 番茄日更4000-6000字是拿全勤的基础
- 30章已有存量可以"定时释放"：前2周每天发2章（约5000字），制造"日更"假象
- 第3周开始需要真正的新增创作跟上

#### 6. 前期反派压迫感加强
- 赵老三的威胁感在前10章不够"恨"
- 建议在前3章就让赵老三做出**实质性的伤害行动**（而不只是嘲笑）

### 🟢 P2：加分项

#### 7. 增加90年代特有细节
- 物价（500元在1992年的购买力描述）
- 通讯方式（没有手机，靠传话、写信）
- 交通方式（拖拉机、自行车、长途汽车）
- 流行文化（收音机、黑白电视、港台歌曲）
- 这些细节能大幅增加"真实感"和"代入感"

#### 8. 每章结尾强化悬念钩子
- 番茄读者"看完一章就走"是常态，结尾钩子决定追读率
- 建议每章结尾设置：悬念/反转/危机预告

## 五、AI写作的合规策略

这是你面临的最大风险点，需要特别注意：

### 5.1 番茄的AI政策现状

| 场景 | 建议 |
|------|------|
| AI生成整章 | 必须勾选"是" |
| AI大幅扩写/改写 | 应当勾选"是" |
| AI润色（小幅） | 灰色地带，但建议勾选 |
| 仅用AI检查错别字 | 可以不勾选 |

### 5.2 你的应对策略

**现实地说**：你的30章有明显的AI辅助痕迹，建议**如实勾选**。

但可以大幅降低风险：
1. **降重后发布**：将AI率从55-61%降到40%以下
2. **勾选AI但强调"AI辅助创作"**：在作者简介中可以说明"人机协作"
3. **保持后续章节的原创性**：如果后续章节人写比例增加，后期勾选"否"也有依据
4. **不要心存侥幸**：番茄有事后检测，不勾选被查出的惩罚远大于如实勾选

### 5.3 勾选AI后如何争取流量

- 番茄不会完全封杀AI辅助作品，但会**降权处理**
- 降权后的作品要靠**内容质量本身留住读者**
- 完读率和追读比如果数据好，算法会逐步恢复推荐
- **关键是前三章必须能留住人**

### 最终结论

> **有人会看，但现在不能直接发。**

《山村小神医》具备在番茄获得稳定读者的基础条件——完整的框架、扎实的设定、清晰的升级线。但当前最大的两个拦路虎是：

1. **AI率偏高**（55-61%）—— 番茄的AI检测是悬在头上的剑
2. **黄金三章不够"抓人"** —— 番茄的算法不等人，前三章留不住人，后面的30万字等于白写

**建议**：花1-2周做好降重和前三章打磨，然后以每周5-7章的速度发布，边发边观察数据，根据数据调整后续创作方向。

这不是"有没有人看"的问题，而是"你准备好了没有"的问题。准备好，就有人看。
