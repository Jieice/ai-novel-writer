# 小说项目启动器

## 触发条件

- 用户说"我想写一本新小说"
- 用户说"帮我创建一个新小说项目"
- 用户说"开新书"
- 用户说"写一本XX类型的小说"

## 工作流程

### 1. 询问小说基本信息

向用户确认：
- **小说类型**：90年代乡土 / 港综同人 / 都市重生 / 其他
- **小说标题**：（可选，可后续生成）
- **核心卖点**：（可选，可后续生成）
- **主角设定**：（可选，可后续生成）

### 2. 创建项目文件夹

```
复制模板：d:\AI\AI小说创作系统\00_项目模板\
重命名为：d:\AI\AI小说创作系统\{小说名称}\
```

### 3. 调用对应技能生成内容

根据小说类型，调用对应的技能：

| 类型 | 技能位置 |
| --- | --- |
| 90年代乡土 | `02_Skill技能库/90年代乡土专用/` |
| 港综同人 | `02_Skill技能库/港综专用/` |
| 通用 | 使用通用技能 |

### 4. 初始化项目文件

创建以下文件（如果模板中没有）：
- `项目文件/小说大纲.md`
- `项目文件/人物设定.md`
- `项目文件/99_当前状态卡.md`
- `项目文件/task_plan.md`
- `项目文件/progress.md`
- `项目文件/pending_hooks.md`
- `项目文件/resource_ledger.md`
- `项目文件/character_relationships.md`
- `项目文件/findings.md`
- `项目文件/file_index.md`

### 5. 输出项目创建报告

```markdown
## 📚 新小说项目已创建

### 项目信息
- **小说名称**：{名称}
- **小说类型**：{类型}
- **项目位置**：d:\AI\AI小说创作系统\{名称}\

### 已创建文件
- [x] 项目文件/小说大纲.md
- [x] 项目文件/人物设定.md
- [x] 项目文件/99_当前状态卡.md
- ...（其他文件）

### 下一步
1. 查看并修改小说大纲
2. 查看并修改人物设定
3. 说"写第1章"开始创作
```

## 示例对话

**用户**：我想写一本90年代乡土小说

**AI**：
好的！我来帮你创建一个90年代乡土小说项目。

请问：
1. 小说标题是什么？（可以说"帮我起一个"）
2. 核心卖点是什么？（如：中医+种田+打脸）

**用户**：标题帮我起一个，卖点是中医秘方+山货生意

**AI**：
[创建项目，生成大纲和人物设定]

## 注意事项

- 如果用户没有提供足够信息，使用默认值或询问
- 项目名称不能与现有项目重复
- 创建后提醒用户检查和修改设定文件
