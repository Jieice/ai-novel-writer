## 一、小说毒点大全

### 1. 人设毒点

| 毒点 | 说明 | 后果 |
| --- | --- | --- |
| 主角圣母 | 无底线原谅敌人、帮敌人 | 读者弃书 |
| 主角窝囊 | 被欺负不反击、忍气吞声 | 读者愤怒 |
| 主角双标 | 对己宽容对人严苛 | 读者厌恶 |
| 主角降智 | 关键时刻变蠢推动剧情 | 读者吐槽 |
| 配角抢戏 | 配角比主角还出彩 | 主角存在感弱 |
| 反派降智 | 反派蠢得不像正常人 | 缺乏紧张感 |
| 女主绿茶 | 表面善良实则心机 | 读者反感 |
| 后宫争宠 | 女主们无脑争风吃醋 | 剧情拖沓 |

### 2. 剧情毒点

| 毒点 | 说明 | 后果 |
| --- | --- | --- |
| 虐主成瘾 | 主角一直被虐不翻身 | 读者憋屈 |
| 虎头蛇尾 | 开局精彩后面崩 | 读者失望 |
| 重复剧情 | 同样套路反复用 | 读者疲劳 |
| 无脑送宝 | 敌人主动送资源 | 缺乏成就感 |
| 强行误会 | 误会推动剧情 | 读者烦躁 |
| 圣母敌人 | 敌人突然变好 | 逻辑崩坏 |
| 金手指滥用 | 金手指解决一切 | 缺乏挑战 |
| 金手指消失 | 金手指被遗忘 | 前后矛盾 |

### 3. 节奏毒点

| 毒点 | 说明 | 后果 |
| --- | --- | --- |
| 开局慢热 | 前三章没钩子 | 读者流失 |
| 水字数 | 大量无意义描写 | 读者跳读 |
| 剧情拖沓 | 一件事写十章 | 读者疲劳 |
| 高潮缺失 | 一直平淡没爽点 | 读者无聊 |
| 结局仓促 | 匆忙收尾 | 读者不满 |

### 4. 逻辑毒点

| 毒点 | 说明 | 后果 |
| --- | --- | --- |
| 前后矛盾 | 前文设定后文推翻 | 读者出戏 |
| 设定崩坏 | 世界观前后不一致 | 读者困惑 |
| 时间线混乱 | 时间顺序错误 | 读者出戏 |
| 人物失忆 | 角色忘记重要信息 | 读者吐槽 |

## 二、冲突设计指南

### 1. 冲突类型

**按强度分：**
- 小冲突：口角、小摩擦（每1-2章一次）
- 中等冲突：利益争夺、小打脸（每3-5章一次）
- 大冲突：生死存亡、大打脸（每10-15章一次）

**按类型分：**
- 打脸冲突：主角被轻视→反击→震惊全场
- 利益冲突：资源争夺、商业竞争
- 生存冲突：生命威胁、绝境求生
- 感情冲突：情敌、误会、选择
- 面子冲突：尊严、名声、地位

### 2. 打脸设计公式

**标准打脸流程：**
```
铺垫（敌人嚣张）→ 挑衅（针对主角）→ 隐忍（主角低调）→ 
爆发（实力展示）→ 震惊（众人反应）→ 后续（敌人下场）
```

**打脸变体：**
- 连环打脸：一次打脸引出更大敌人，再次打脸
- 借力打脸：借他人之手打脸
- 反差打脸：低调者突然高调
- 预判打脸：提前布局，敌人自投罗网

**打脸节奏：**
- 小打脸：1章内完成
- 中打脸：2-3章完成
- 大打脸：5-10章铺垫+爆发

### 3. 情绪调动技巧

**愤怒情绪：**
- 敌人嚣张跋扈
- 主角被无理欺负
- 弱者被欺凌
- 不公正待遇

**爽感情绪：**
- 实力碾压
- 打脸成功
- 获得认可
- 收获资源

**期待情绪：**
- 悬念钩子
- 伏笔埋设
- 实力隐藏
- 即将爆发

**紧张情绪：**
- 生死危机
- 时间紧迫
- 资源不足
- 敌人强大

## 注意事项

### 毒点避免原则
- 主角必须爽，不能憋屈
- 敌人必须蠢得合理
- 打脸必须爽得彻底
- 节奏必须张弛有度

### 冲突设计原则
- 小冲突不断，大冲突有节奏
- 每次冲突都要有爽点
- 冲突解决要有成就感
- 不能让读者太累

### 情绪调动原则
- 先抑后扬是王道
- 愤怒积累要足够
- 爽感释放要彻底
- 期待感要持续
