# 参考书风格学习技能

## 触发条件

- 用户说"学习风格"
- 用户说"分析参考书"
- 用户说"提取风格"
- 用户说"风格学习"
- 用户说"学习这本书的写法"

## 工作流程

### 1. 选择参考书

从 `01_参考小说库/` 目录选择要学习的参考书：
- 按类型分类：乡土历史、港综、同人、综武等
- 选择与当前项目类型匹配的参考书
- 或由用户指定特定参考书

### 2. 多维度分析

#### 2.1 语言风格分析

**词汇层面：**
- 高频词汇统计
- 口语化程度分析
- 方言/俚语使用
- 时代特色词汇

**句式层面：**
- 平均句长
- 长短句比例
- 句式结构特点
- 标点使用习惯

**段落层面：**
- 段落长度分布
- 段落开头方式
- 场景切换手法

#### 2.2 叙事节奏分析

**开篇节奏：**
- 第一章信息量
- 开篇钩子设计
- 背景交代方式

**推进节奏：**
- 每章字数分布
- 剧情推进速度
- 高潮低谷分布

**收尾节奏：**
- 章节结尾钩子
- 悬念设置手法
- 伏笔埋设频率

#### 2.3 人物塑造分析

**主角塑造：**
- 出场方式
- 性格展示手法
- 成长轨迹设计
- 金手指使用方式

**配角塑造：**
- 配角功能分类
- 配角出场频率
- 配角与主角关系构建

**反派塑造：**
- 反派出场时机
- 反派层次设计
- 打脸节奏安排

#### 2.4 冲突设计分析

**冲突类型统计：**
- 打脸冲突
- 利益冲突
- 感情冲突
- 生存冲突

**冲突强度分析：**
- 小冲突频率
- 中等冲突间隔
- 大高潮铺垫

**冲突解决方式：**
- 智力碾压
- 武力解决
- 人脉压制
- 金手指助力

将风格指南保存到：
- 项目目录：`项目文件/风格指南.md`
- 通用库：`02_Skill技能库/{类型}专用/风格指南_{参考书名}.md`

## 使用示例

### 学习特定参考书
```
"学习《港综-开局拒绝卧底》的风格"
"分析参考书《乡村那些事儿》的写法"
```

### 学习某类型的多本书
```
"学习港综类参考书的共同风格"
"分析乡土类小说的写作特点"
```

### 应用到当前项目
```
"把学到的风格应用到当前项目"
"按照参考书的节奏调整我的小说"
```

## 注意事项

### 学习原则
- 学习的是写作技巧，不是抄袭内容
- 提取的是模式，不是具体情节
- 借鉴的是节奏，不是照搬结构

### 应用原则
- 结合自己项目特点调整
- 不要生搬硬套
- 保持原创性

- 参考书仅供学习分析
- 不复制具体内容
