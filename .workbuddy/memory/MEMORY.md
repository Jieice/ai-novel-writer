## AI检测研究结论（2026-04-22）

### 检测原理
- **Perplexity（困惑度）**：AI文本低困惑度（可预测），人类文本高困惑度（用词意外）
- **Burstiness（突发性）**：AI文本句长均匀，人类文本长短交替
- **词频分布**：AI倾向高频正式词（"缓缓地""微微""不禁"），人倾向口语/不标准搭配
- **统计指纹**：AI在标点分布、段落长度、过渡词频率上高度一致

### 降重有效性验证（2026-04-22 天眼AI实测后修正）
- 口语化改造→❌ 朱雀实测无效（CH1深度重写后仍100%）
- 感官细节→❌ 加了嗅觉/触觉/听觉，朱雀仍判100%
- 节奏打散→❌ 长短句交替+断裂式过渡，朱雀仍判100%
- 删AI模板词→❌ 不影响朱雀判定
- **第一人称+8条铁律→✅ 天眼AI实测有效，AI率明显降低**
- 引号格式修复→非检测因素但影响阅读体验 ✅

### 平台现状
- 番茄/朱雀等平台AIGC容忍阈值已降至15-20%
- 跨章高频重复短语是AI痕迹最大标志（"心里咯噔一下""冷笑""微微上扬"等）
- 静态检测器（如GPTZero）准确率约60-70%，假阳性较高
- 番茄自有检测机制比公开工具更严格

### 降重策略总结（2026-04-22 天眼AI实测后修正）
1. **✅ 已验证有效**：风格指南8条铁律+第一人称写法，天眼AI检测AI率明显降低
2. **可能有效**：口述+转录（语音输入让叙事逻辑变"乱"），然后AI润色
3. **AI自重写（第三人称）无效**：无论深度重写还是表面修补，朱雀都能识别（实测AIGC=0.9984）
4. **无效做法**：同义词替换、改标点、删模板词、打断句长、加感官（全被朱雀识破）
5. **关键认知**：不是"改词"是"换写法"——第一人称+人味铁律=改变写作方式本身
6. **免费检测平台**：天眼AI（tianyanai.org），替代朱雀
