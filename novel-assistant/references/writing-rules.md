# 写作技巧参考(工具书,非规则)

> 本文件是**按需查阅的参考资料**,不是硬规则。硬规则看 `CLAUDE.md` 的 7 条铁律。
> 这里的内容是写作技巧、类型词汇、医学和年代底线,帮助把字写顺、写真、写活。

---

## 一、核心写作原则

**展示,不讲述**:
- 用动作、反应、后果、语气、细节代替抽象判断
- 不写"他很愤怒",写"他捏碎了茶杯,滚烫的茶水流过指缝,但他像没感觉一样"

**盐溶于汤写法**:
- 主角的野心、价值观、狠劲、欲望都要化进做事方式里,不靠口号喊
- 反例:"俺一定要出人头地!"(口号)
- 正例:"他把那张被撕下来的名片揣进怀里,指头把纸角捏了又捏"(做事方式)

**朴素讲故事模式**:
- 让读者觉得"这就是在讲一个故事",不是"这是在写小说"
- 避免排比、对仗、过度修辞

## 二、BypassAIGC 降重四阶段(核心工具)

### 阶段一:润色(基础降重)

**1. 动词短语扩展**:
- "说" → "开口说了一句" / "轻声应了" / "冷笑" / "哼了一声"
- "看" → "抬起头看了一眼" / "目光扫过" / "眯起眼睛" / "盯着瞧"
- "走" → "迈步走了过去" / "快步走来" / "慢悠悠地蹭过去"
- "想" → "心里琢磨着" / "脑子里冒出个想法" / "寻思"
- "感觉" → "心里一沉" / "头皮发麻" / "后脊梁窜过一股凉气"

**2. 环境细节增强**:
- 场景感官:温度、光线、声音、气味
- 动作小细节:手指敲击、眼神变化、呼吸节奏
- 时间标记:当时是几点、天气如何

**3. 逻辑辅助词**:
- 适度加:的、地、着、了、吧、嘛、呢、呀
- 每段 1-2 个即可,不堆砌
- 用"于是/然后/结果/后来"连接

**4. 句式变化**:
- 长句拆短,短句补长
- 被动变主动,主动加描述
- 删冗余连接词

### 阶段二:去 AI 味(深度)

**1. 口语化表达**:
- 语气词:呢/吧/嘛/咋/啥/呗/哩/哟
- 每段 1 个就够,**不要行尾全挂**(CH22 教训)
- 停顿用 "——" / "、"

**2. 打破 AI 对称结构**:
- AI 爱 A-B, C-D 对称;真人写 A-B, D-C(打乱)
- 不要结构过于工整,适当不对称

**3. 加跳跃感**:
- 真人思维会跳,用"突然/不过/话说回来"过渡
- 不要每句都有逻辑联系

**4. 类型特色表达**:见第三节词汇表

### 阶段三:特殊场景降重

**对话场景**:
- 加动作描写(说话时的动作、表情、神态)
- 加环境反应(别人听后的反应、背景声音)
- 用动作打断对话,不一口气说完

**心理活动**:
- 不直接"他觉得很..."
- 具象化:"心沉下去" / "手心出汗" / "嗓子眼堵"
- 用往事触发情绪

**描写场景**:
- 多感官:视+听+嗅+触+味
- 时空跳跃:先快速带过,再详细写
- 人物反应:通过人物的眼睛看世界

### 阶段四:整体润色

- 删过度工整的排比句
- 删过多连接词
- 删过度完美的逻辑链
- 加不完美的表达、真人习惯的重复

### 阶段五:Humanizer 结构性模式检测(新增)

BypassAIGC 四阶段解决词级问题,但 AI 的**结构性套路**需要额外扫描:

**高优先级检测(A 类——立即修改)**:
- **否定式排比**: `不是……也不是……更不是……`、`既不……也不……`
- **三段式堆砌**: `首先/其次/最后`、每件事都凑"A、B、C"
- **-ing 结尾虚假分析**: 句末以`突出……`、`彰显……`、`为……做出贡献`结尾
- **谄媚语气**: `当然/毫无疑问/无可否认`、`您的观点非常正确`
- **强行展望结尾**: `挑战与机遇并存`、`光明的前途在等待着……`

**中优先级检测(B 类——影响观感)**:
- **填充短语**: `总而言之`、`值得注意的是`、`实际上`、`从某种意义上说`(每章 > 2 次 = 啰嗦)
- **AI 高频词**: `此外`(>2次/章)、`深入探讨`(>1次/章)、`至关重要`(>1次/章)
- **过度限定**: `可能`、`也许`、`大概`、`应该`(每章 > 3 次 = 不自信)
- **虚假范围**: `从 X 到 Y` 但 X/Y 无实际意义
- **刻意换词**: 同一事物(药物/药品/药剂、患者/病人)反复换名

**低优先级检测(C 类——格式问题)**:
- 破折号 `——` 每章 > 3 次
- 正文有 `**加粗**` 或 `*斜体*`
- 对话用了 `「」` 而非 `""`
- 标题/列表有 `🎯🔥⭐` 等表情符号

> 详细模式和改写原则见 `references/humanizer-patterns.md`

## 四、情感表达

**不直接写情绪,用细节承载**:
- ❌ "他很愤怒" → ✅ "他捏碎了茶杯"
- ❌ "她很伤心" → ✅ "她盯着那张照片看了很久,眼眶红了"
- ❌ "他很紧张" → ✅ "他的手心全是汗"

**情绪递进有层次**:
- 先铺垫 → 再积累 → 再爆发
- 不要一上来就情绪顶格

## 五、代入感六大支柱

1. **基础信息与标签化**:让读者尽快知道主角是谁、在哪里、什么状态
2. **具体化**:尽量写读者脑海里能浮现的东西
3. **可视化与熟悉感**:场景最好能让人"看见"
4. **认知共鸣**:主角的选择让目标读者觉得"对,我也会这么想"
5. **情绪共鸣**:通过压力、羞辱、期待、愤怒、贪心、温情、恐惧带动情绪
6. **五感代入**:关键处加入声音、气味、温度、触感、味道

## 六、人物构建六步法

1. 搞清这个人物在故事里承担什么功能
2. 明确背景、阶层、欲望、恐惧、伤口、骄傲、资源、当前利益
3. 决定这些东西如何影响他的说话节奏、动作习惯、风险偏好
4. 加反差、加阴影、加灰度,拒绝脸谱化
5. 让人物变化绑定到具体事件,不空转
6. 每遇到重要情节都回头复查人设有没有偏

**角色行为公式**:`过往经历 + 当前利益 + 性格底色`
**活人公式**:`核心标签 + 反差细节`

## 七、配角独立化(避免工具人)

每个有用配角必须能回答:

1. **他想要什么?**(明确的目标)
2. **他害怕什么?**(底线/弱点)
3. **他的立场是什么?**(在冲突中选哪边)
4. **他为自己的利益做了什么?**(不是帮主角,是帮自己)
5. **如果主角不在场,他会做什么?**(独立于主角的剧情)

**示例:林秀梅的独立线**
- 她是代课老师,面临转正考核压力(自己的困境)
- 家里反对她跟陈大山在一起(家庭压力)
- 她帮陈大山是因为想证明自己(自己的算盘)

**示例:赵老三的深度**
- 家以前穷被看不起,现在有钱想报复(过往创伤)
- 想垄断后山药材(利益动机)
- 害怕村民看不起他,所以必须装强大(怕的东西)

## 八、反派话术(高级 vs 低级对照)

**❌ 低级(混混型,禁用)**:
- "你小子别不识抬举!"
- "咱们走着瞧!"
- 气急败坏指着鼻子骂

**✅ 高级(笑面虎型,推荐)**:
- 表面客气:"大山兄弟有骨气是好事。但做生意光有骨气可不够。"
- 绵里藏针:"俺们恒昌背后可是有人的,县城都有路子。"
- 笑里藏刀:"山路不好走,以后可得当心脚下。"

**身份匹配**:
- 镇上药商/县城老板 → 说话体面,不挂"呗/嘛"
- 村里泼妇 → 可粗俗,可"俺"自称

## 九、反派实质行动清单(六选一写入本章)

避免"只威胁不行动"的纸老虎反派:

- 买通镇上药房拒收主角的药
- 带人毁掉主角的药田
- 向大队举报主角"搞封建迷信"
- 散布谣言说主角的药吃死人
- 威胁村民不准找主角看病
- 截断主角上山/进镇的路
- 拉拢主角身边的人

## 十、医学常识底线(乡土中医文)

这是中医文不是修仙文。治病必须符合规律。

| 疾病 | 合理见效时间 | 不合理的表现 |
|---|---|---|
| 感冒/风寒 | 2-3 天退烧 | ❌ 半天痊愈 |
| 腰腿疼/风湿 | 1-2 周缓解 | ❌ 一夜全好 |
| 慢性胃病 | 2-4 周好转 | ❌ 三天走路 |
| 外伤 | 1-2 周愈合 | ❌ 第二天活蹦乱跳 |
| 皮肤病 | 2-4 周见效 | ❌ 抹了就好 |

**灵珠加速规则**:
- 只能缩短到合理下限(风湿 1 周 → 3 天有缓解,不能一夜)
- 加速后必有恢复期描述(虽然好了但还是虚)
- 别人对"好得快"的反应应是**怀疑**而非直接吹捧

**治病场景规则**:
- 每次治病必写**诊断过程**(望闻问切,不是一眼就看出来)
- 治疗方案要有**具体方法**(药方/针灸/推拿)
- 病人必有**复诊**或**随诊**描述
- 治病过程中要有**波折**(药不对症需要调整、病人不配合、药材不够)

## 十一、90 年代时代质感

不能只写"1992 年"和"大米三毛八",要写出生活质感:

| 维度 | 要求 | 示例 |
|---|---|---|
| 经济氛围 | 下海潮、万元户、个体户刚起步 | "村东头老张家去年倒腾粮食,据说攒了三千块" |
| 社会心态 | 对"能人"既羡慕又忌惮 | "有人背后嚼舌根,说陈家小子不务正业" |
| 人情世故 | 托关系、走后门、人情债 | "没有大队长的条子,镇上供销社不批货" |
| 基础设施 | 交通不便、通讯靠信/口信 | "从村里到镇上骑自行车一个钟头,赶上下雨天路都走不了" |
| 治安环境 | 派出所人少、宗族影响大 | "赵老三他二舅在公社当干事,村里人不敢惹他" |
| 思想观念 | 保守、迷信、重男轻女 | "女孩子读那么多书干啥?不如早点嫁人" |

**细节写入规则**:每 3 章至少有 1 处体现 90 年代特有社会氛围的细节。

## 十二、NPC 态度渐变(避免翻脸比翻书快)

群体态度不能一步到位,必须有人带头、有人观望、有人质疑。

| 阶段 | 表现 | 过渡条件 |
|---|---|---|
| 排斥/怀疑 | "这小子能行?" | - |
| 观望 | "先看看再说" | 1-2 次成功案例 |
| 半信半疑 | "好像有点本事" | 3 次以上成功+口碑传播 |
| 认可 | "大山有两下子" | 亲眼见证+利益相关 |
| 支持 | "大山的事就是咱的事" | 多次受益+感情积累 |
| 狂热 | "大山说啥就是啥" | 集体利益绑定 |

**每个 NPC 至少经过 3 个阶段才能从排斥到支持**,必须有具体事件触发。

更新 `character_states.md` 时参考(范围 -100 到 +100):

| 事件类型 | 变化 | 示例 |
