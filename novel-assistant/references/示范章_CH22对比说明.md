## 创作策略对比

### 剧情取舍

| 原 CH22 场景 | 示范版 CH22 | 说明 |
|---|---|---|
| 1. 陈大山醒来 | **保留,扩写**(挫折+代价) | 单场戏展开 |
| 2. 镇长归来 | **并入场景 1** | 赵德明态度转变用 1 个眼神带过 |
| 3. 治疗完成 | **并入场景 1** | 扎针戏作为挫折点 |
| 4. 恒昌吴经理登门(威胁) | **保留,作为第 2 场戏** | 铺垫+绵里藏针 |
| 5. 回村发现被抢 | **❌ 砍掉,挪到 CH23 开头** | 原章节奏垮的元凶 |

**核心思想**：**一章最多 2 场戏**,每场戏展开写透。原 CH22 的崩溃就是因为把 5 场戏塞 3137 字,每场都是草稿状态。

### 中二独白 → 动作细节

**原 CH22(3 次崩塌)**:
```
"等着吧,俺迟早让你赵老三十倍奉还！"
他攥紧车把,心里头发狠。
```

**示范版**(0 次,用动作替代):
```
陈大山过去,伸手把名片按住。
他没急着揉,也没急着扔。他拈着名片看了看上头"恒昌药行 吴德明"几个字,
又翻过来看了看背面——空白的,啥都没有。
然后他把名片对折。
再对折。
塞进了贴身里衬的口袋里,跟那本百草手札放在一块儿。
```

**读者感受**:
- 喊"十倍奉还"是**怨气**,读者审美疲劳
- 把名片对折、对折、再揣进手札口袋——是**克制的狠**,读者记得住这个动作

### 对话节奏对比

**原 CH22**(每句挂语气词):
```
"醒了？"周济生的声音从旁边传来呢。
"周老……"他撑着身子想坐起嘛。
"别动嘛。"
"俺行医这么多年,还没见过哪个年轻人像你这么拼的呢。"
```

**示范版**(动作+直说+语气词混用):
```
"慢着。"
周济生的声音从门边过来。老头端了个粗瓷碗进屋,碗里是小米粥,冒着热气。
"俺守你守了半宿。"
周济生把碗搁在床头柜上。
"你小子这身子,怕是三天都缓不过劲儿来。"
```

**差别**:
- 对话行尾语气词从 63% 降到 9%(脚本实测)
- 中间穿插动作描写(端碗、搁在柜上),让对话有呼吸感
- 语气词用在叙述/关键句,不是每句都挂

### 反派话术升级

**原 CH22**(混混型):
```
"大山兄弟,你这就认怂了？"
"哎——这就走了？"
```

**示范版**(笑面虎型,吴德明镇上药商身份):
```
"山路崎岖,货要顺顺当当从村里运到镇上,这中间总得有人照应。
陈大夫,您说是不是这个理？"
"陈大夫,路上走稳啊。"
```

**差别**:
- 镇上药商说话体面客气,不说"呗/嘛"
- 威胁是绵里藏针("山路崎岖"、"路上走稳"),不是直接喊话
- 主角读懂威胁但没当场发作——身体虚+对方有赵德明在场,克制就是**人际挫折**

## 给 Trae 的参考要点

1. **先想场景数,再想字数**——一章 2 场戏是上限,不是建议
2. **中二独白用动作替代**——主角受委屈想喊"等着吧",改成一个具体的手势/物件细节
3. **语气词用在合适地方,别挂行尾**——"嘛/呗/呢"用在叙述和关键句,对话用动作带
4. **反派话术跟身份走**——镇上药商体面,村里泼妇粗俗,不要全都"你小子"
5. **金手指代价要具体**——不要模糊说"虚弱",要说"胳膊发抖/腿一软/冷汗顺脊梁流"这种关键词(脚本识别)
