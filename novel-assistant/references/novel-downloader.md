# 小说自动下载功能

## 功能说明

从ffxs8.com网站下载小说txt文本，用于学习参考或获取素材。

## 工作流程

### 1. 询问用户

用户说"下载小说"或"获取素材"时触发。

询问：
- 书名或搜索词
- 下载全部还是部分章节
- 保存到哪个类型目录

### 2. 访问网站

**已知信息：**
- 网站首页可访问：`https://ffxs8.com`
- 显示最新更新的小说列表
- 搜索URL格式待确认（`search.aspx`返回404）

### 3. 尝试搜索

如果用户提供了书名，尝试以下URL格式：
```
https://ffxs8.com/search.aspx?searchkey={书名}
https://ffxs8.com/s/{书名}
https://ffxs8.com/book/{书名}
```

### 4. 获取小说信息

解析搜索结果页面或首页列表，获取：
- 书名
- 作者
- 简介
- 章节列表URL

### 5. 访问小说详情页

获取完整章节列表：
- 每章标题
- 每章下载链接

### 6. 下载章节

逐章下载txt内容，处理编码：
- 尝试UTF-8
- 如乱码则尝试GBK
- 保存到本地

### 7. 保存到参考库

保存路径：
```
d:/AI/AI小说创作系统/01_参考小说库/{类型}/{书名}/第X章.txt
```

## 技术实现

### 网站结构

ffxs8.com是一个小说聚合网站，支持按书名搜索。

**注意**：网站URL结构可能变化，如遇404需更新URL格式。

### 编码处理

```python
# 尝试UTF-8
try:
    content = response.content.decode('utf-8')
except:
    # 尝试GBK
    content = response.content.decode('gbk')
```

### 章节解析

使用正则表达式解析HTML：
```python
# 获取章节链接
```

```markdown

### 失败原因
- 网站URL结构变化
- 书名不存在
- 编码错误

### 建议
- 检查书名是否正确
- 换个搜索词试试
- 稍后再试
- 手动访问网站确认URL格式
```

## 注意事项

1. 网站可能有访问频率限制
2. 部分小说需要登录才能下载
3. 尊重版权，仅用于学习参考
4. 如网站结构变化，需更新解析规则
