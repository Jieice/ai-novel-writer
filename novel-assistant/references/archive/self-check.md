# 自检清单

## 写作前自检

- [ ] 凡是会变动的现实信息，我查过了吗？
- [ ] 如果是风格学习任务，我先做量化分析了吗？
- [ ] 如果是风格学习任务，结论转成后续可复用的写作默认值了吗？

## 写作中自检

- [ ] 回复语气有跟着用户的真实语气走吗？还是又回到了固定助手腔？
- [ ] 如果是文本优化任务，保住作者原来的个性了吗？
- [ ] 如果是文本优化任务，按用户要求控制长度了吗？
- [ ] 如果是文本优化任务，有讲清为什么这样改吗？

## 写作后强制检查（必须全部通过才能输出）

### 降重质量检查

- [ ] 每段是否应用了至少两个阶段的降重策略？
- [ ] 是否存在超过3句话没有动作/对话/感官描写的纯叙述段？
- [ ] 全文是否有超过5个逻辑连接词（但是/因此/所以/然而/此外）连续出现？
- [ ] 是否有对称句式超过2组？
- [ ] 口语化词汇（呢/吧/嘛/咋/啥）是否每段至少出现1个？
- [ ] 是否有连续2段以上没有对话或动作？
- [ ] **反派对话是否符合其社会身份**？镇上药商不说"呗/嘛"，村里泼妇可以粗俗

### 内容质量检查

- [ ] 本章是否推动了至少一个伏笔或主线？
- [ ] 章末是否有钩子？
- [ ] 本章是否至少有一个冲突点或紧张点？
- [ ] 连续3章以上没有冲突了吗？（如果是，必须加冲突）
- [ ] 人物行为是否符合人设（过往经历+当前利益+性格底色）？

### 反重复/反套路检查（必须通过）

- [ ] 是否使用了黑名单中的禁用短语？（心里头有些异样的感觉/陈大山笑了笑/消息传得比风还快等）
- [ ] 是否有重复的结构模式？（走进院子→众人围过来→坐下→对话→回答）
- [ ] 是否连续2章用了同样的"采药→发现珍稀药材→用灵珠"模式？
- [ ] 代词使用是否多样化？（避免全程只用主角名和女主名）
- [ ] 本章是否引用了前几章的对话或情节？（如果不是故意呼应，就是重复）
- [ ] 本章的结尾是否与前几章结尾模式雷同？（都安排悬念/对话收尾等）
- [ ] **是否出现了F-002/F-003等伏笔代号或系统提示词**？（打破第四面墙）

### 场景多样性检查

- [ ] **对话场景是否全在同一地点**？（如有，必须拆分场景）
- [ ] 本章是否有场景切换（院内→村口→河边→镇上）？
- [ ] 是否避免了"NPC排队上门"的站桩模式？
- [ ] 主角是否有**亲眼目睹冲突**的场景，而非只是听别人说？

### 角色物理状态一致性检查

- [ ] 主角虚弱时是否始终保持了虚弱状态（扶着墙/腿软/眼前发黑）？
- [ ] 是否有"上一秒病重下不来床，下一秒精神抖擞"的状态割裂？

### 主角挫折检查（必须通过）

- [ ] 本章主角是否经历了挫折、失败或付出代价？
- [ ] 如果本章是**建立期(Ch1-3)**、**成功庆祝章(Ch9/Ch13)**：允许无重大挫折
- [ ] 如果本章是**冲突升级期(Ch11-14)** 或 **反击期(Ch15+)**：必须有挫折
- [ ] 如果本章全是顺风顺水，有没有其他章节作为缓冲？
- [ ] 主角的爽是付出了什么代价换来的？
- [ ] 本章是否避免了"做事→成功→完美收尾"的单向模式？

### 配角独立化检查（必须通过）

- [ ] 主要配角（赵老三、林秀梅、刘瘸子等）有明确的目标吗？
- [ ] 主要配角有害怕的东西或底线吗？
- [ ] 主要配角的行动有自己的逻辑吗？（不完全是帮主角）
- [ ] 如果主角不在场，这个配角还会做类似的事吗？
- [ ] 本章是否推进了配角的独立剧情线？（不是只是围着主角转）

### 角色漂移检查

- [ ] 主要角色的行为是否符合 character_states.md 中的性格锚点？
- [ ] 是否有角色突然表现出与锚点矛盾的性格？
- [ ] 说话节奏和动作习惯是否前后一致？

### 格式检查

- [ ] 段落是否控制在1-3句？
- [ ] 对话是否独立成段？
- [ ] 对话是否使用了引号 `"`？（严禁用逗号或冒号代替）
- [ ] 场景切换是否有空行分隔？

### 反派升级检查

- [ ] 本章反派是否有实质性行动？
- [ ] 反派行动是否升级（比上章更狠）？
- [ ] 反派是否至少3次成功阻击主角（不能全是口头威胁）？
- [ ] 反派失败后是否升级了手段？
- [ ] **反派是否是"笑面虎"类型**（笑着威胁，不气急败坏）？
- [ ] **反派话术是否符合其社会身份**（镇上商人→体面客气；村里泼妇→粗俗直接）？

### NPC态度渐变检查

- [ ] 本章态度转变的NPC是否经过了至少3个阶段？
- [ ] 态度转变是否有具体事件触发？
- [ ] 群体态度是否有"有人带头、有人观望、有人质疑"的层次？
- [ ] 是否避免了翻脸比翻书快的硬编码转变？

### 金手指代价检查

- [ ] 使用金手指后是否产生了真实代价？
- [ ] 代价是否在后续剧情中产生了影响？
- [ ] 灵珠是否在关键时刻用完了，导致只能靠真本事？
- [ ] 治病是否总是成功，没有治坏过？

### 医学常识检查

- [ ] 疾病的见效时间是否符合中医规律？
- [ ] 是否有合理的诊断过程和治疗方案？
- [ ] 病人是否有复诊/随诊的描述？
- [ ] 治病过程中是否有波折？

### 时代质感检查

- [ ] 本章是否有1处以上体现90年代特有社会氛围的细节？
- [ ] 人物的思维方式、办事方式是否是那个年代的？
- [ ] 是否只是简单替换现代词（如"手机"→"电话"）？
- [ ] 是否有体现下海潮、人情世故、保守思想等时代特征？

## AI去味自检

- [ ] 去掉的是机械感，而不是改坏原意吗？
- [ ] 有没有矫枉过正，乱加俚语、乱加情绪、乱加装饰？
- [ ] 如果用了第二人称，统一使用了"你"吗？
- [ ] 追求真实自然，而不是打着"规避检测"的旗号乱改？

## 朴素讲故事模式自检

- [ ] 读者会觉得这个作者挺有意思吗？
- [ ] 语言自然流畅吗？
- [ ] 这是有故事感的文字，还是只是散文感、任务感、汇报感？

## 人物自检

- [ ] 人物行为符合"经历+利益+性格底色"吗？
- [ ] 关键人物都还聪明、还自私、还像活人吗？
- [ ] 人设前后是否一致？有没有崩塌？

## 细节自检

- [ ] 关键地方有足够具体的时代细节和感官细节吗？
- [ ] 这一场、这一章真的推动了主线、人物、气氛或伏笔吗？
- [ ] 删掉废话、套话、重复表达和作者自嗨了吗？

## 系统与结尾自检

- [ ] 系统有帮助，但没有抢走主角的努力和判断吗？
- [ ] 结尾有余劲吗？能把人往后拽吗？
- [ ] 这段内容还符合现实向世界和起始时间线吗？

## 最终确认

- [ ] 如果发现问题，我真的改掉了吗？还是只是看见了问题？
