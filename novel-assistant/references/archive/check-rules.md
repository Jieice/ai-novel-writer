# 检查规则

## 毒点检测（18个雷点）

### 人设毒点

| 毒点 | 后果 |
| --- | --- |
| 主角圣母 | 读者弃书 |
| 主角窝囊 | 读者愤怒 |
| 主角双标 | 读者厌恶 |
| 主角降智 | 读者吐槽 |
| 配角抢戏 | 主角存在感弱 |
| 反派降智 | 缺乏紧张感 |
| 女主绿茶 | 读者反感 |
| 后宫争宠 | 剧情拖沓 |

### 剧情毒点

| 毒点 | 后果 |
| --- | --- |
| 虐主成瘾 | 读者憋屈 |
| 虎头蛇尾 | 读者失望 |
| 重复剧情 | 读者疲劳 |
| 无脑送宝 | 缺乏成就感 |
| 强行误会 | 读者烦躁 |
| 金手指滥用 | 缺乏挑战 |

### 节奏毒点

| 毒点 | 后果 |
| --- | --- |
| 开局慢热 | 读者流失 |
| 水字数 | 读者跳读 |
| 剧情拖沓 | 读者疲劳 |
| 冲突缺失 | 读者无聊 |
| 信息轰炸 | 读者困惑 |

## 毒点避免原则

- 主角必须爽，不能憋屈
- 敌人必须蠢得合理
- 打脸必须爽得彻底
- 节奏必须张弛有度

## 战力跟踪

| 检查项 | 说明 |
| --- | --- |
| 能力变化 | 主角/配角是否有能力变化 |
| 变化合理性 | 能力变化是否有合理解释 |
| 战力崩坏 | 是否出现突然变强/变弱 |

**战力等级参考：**

| 等级 | 说明 |
| --- | --- |
| LV1 | 普通人，无特殊能力 |
| LV2 | 略有特长，懂医术、会武功 |
| LV3 | 小有所成，能解决一般问题 |
| LV4 | 独当一面，能应对中等挑战 |
| LV5 | 高手级别，能解决重大问题 |

**更新文件**：`项目文件/power_tracking.md`

## 时间线跟踪

**记录格式：**
```markdown
| 章节 | 时间 | 事件 | 主要人物 | 天气 | 地点 |
| --- | --- | --- | --- | --- | --- |
| 第1章 | 1990年春 | 主角出场 | 陈大山 | 晴 | 山村 |
```

**检查内容：**

| 检查项 | 说明 |
| --- | --- |
| 时间连续性 | 时间是否连续推进 |
| 时间冲突 | 是否有时间矛盾 |
| 天气一致 | 天气是否合理变化 |

**更新文件**：`项目文件/timeline.md`

## 质量检查

| 维度 | 检查内容 | 警告阈值 |
| --- | --- | --- |
| 风格一致 | 对比前章风格特征 | 差异>30% |
| 节奏合理 | 冲突间隔是否合理 | 间隔>5章无冲突 |
| 字数稳定 | 字数是否在目标范围 | 偏差>20% |

**更新文件**：`项目文件/quality_report.md`

## 一致性扫描

| 检查项 | 说明 |
| --- | --- |
| 人物OOC | 人物行为是否符合人设 |
| 设定一致 | 世界观设定是否一致 |
| 伏笔推进 | 伏笔是否有推进/回收 |
| 资源连续 | 获得的资源是否连续出现 |

**更新文件**：`项目文件/consistency_report.md`

## 防偏航检查

| 检查项 | 说明 |
| --- | --- |
| 剧情偏离 | 是否偏离大纲主线 |
| 节奏偏离 | 是否偏离节奏规划 |
| 人物偏离 | 人物发展是否符合规划 |

**偏航等级：**

| 等级 | 偏离度 | 处理 |
| --- | --- | --- |
| 正常 | <10% | 无需处理 |
| 轻微 | 10-30% | 提示注意 |
| 中等 | 30-50% | 建议修正 |
| 严重 | >50% | 强烈建议修正 |

**更新文件**：`项目文件/deviation_report.md`

## 冲突设计原则

- 小冲突不断，大冲突有节奏
- 每次冲突都要有爽点
- 冲突解决要有成就感
- 不能让读者太累
