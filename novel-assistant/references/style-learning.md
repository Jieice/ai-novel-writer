# 风格学习

## 工作流程

### 1. 选择参考书

从 `01_参考小说库/` 目录选择要学习的参考书：
- 按类型分类：乡土历史、港综、同人、综武等
- 选择与当前项目类型匹配的参考书
- 或由用户指定特定参考书

### 2. 多维度分析

#### 2.1 语言风格分析

**词汇层面：**
- 高频词汇统计
- 口语化程度分析
- 方言/俚语使用
- 时代特色词汇

**句式层面：**
- 平均句长
- 长短句比例
- 句式结构特点
- 标点使用习惯

**段落层面：**
- 段落长度分布
- 段落开头方式
- 场景切换手法

#### 2.2 叙事节奏分析

**开篇节奏：**
- 第一章信息量
- 开篇钩子设计
- 背景交代方式

**推进节奏：**
- 每章字数分布
- 剧情推进速度
- 高潮低谷分布

**收尾节奏：**
- 章节结尾钩子
- 悬念设置手法
- 伏笔埋设频率

#### 2.3 人物塑造分析

**主角塑造：**
- 出场方式
- 性格展示手法
- 成长轨迹设计
- 金手指使用方式

**配角塑造：**
- 配角功能分类
- 配角出场频率
- 配角与主角关系构建

**反派塑造：**
- 反派出场时机
- 反派层次设计
- 打脸节奏安排

#### 2.4 冲突设计分析

**冲突类型统计：**
- 打脸冲突
- 利益冲突
- 感情冲突
- 生存冲突

**冲突强度分析：**
- 小冲突频率
- 中等冲突间隔
- 大高潮铺垫

**冲突解决方式：**
- 智力碾压
- 武力解决
- 人脉压制
- 金手指助力

### 4. 保存风格指南

将风格指南保存到：
- 项目目录：`项目文件/风格指南.md`
- 通用库：`02_Skill技能库/{类型}专用/风格指南_{参考书名}.md`
