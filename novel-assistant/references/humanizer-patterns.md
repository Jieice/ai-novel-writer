# Humanizer 24 模式速查表

> 基于 Wikipedia "Signs of AI writing" 指南，适配中文网文场景。
> 写作时对照自查，检查时脚本新增 WARNING 提示。

---

## 一、内容模式（6种）

### 1. 过度强调意义/遗产/趋势
**特征词**：作为/充当、标志着、见证了、是……的体现/证明/提醒、极其重要/至关重要/核心作用、凸显/强调/彰显其重要性、反映更广泛的趋势、为……奠定基础

**问题**：AI 通过添加"某事如何代表/促进更广泛主题"的陈述来夸大重要性

**改写原则**：删掉意义升华句，直接说事实

### 2. 过度强调知名度/媒体报道
**特征词**：独立报道、国家媒体、由知名专家撰写、活跃的社交媒体账号、据……报道

**问题**：反复强调知名度主张，列出来源却不给上下文

**改写原则**：只引用具体有上下文的内容

### 3. -ing 结尾的肤浅分析
**特征词**：突出……、强调……、彰显……、确保……、反映……、为……做出贡献、培养/促进……、涵盖……、展示……

**问题**：AI 在句末加现在分词短语增加虚假深度

**改写原则**：删掉句末 -ing 分词结构，保留主句信息

### 4. 宣传/广告式语言
**特征词**：无缝体验、直观且强大、充满活力、引领未来、卓越品质、创新解决方案

**问题**：营销套话取代具体描述

**改写原则**：用具体功能/数字/真实反馈替代抽象形容词

### 5. 模糊归因
**特征词**：专家认为、行业报告显示、研究表明、权威人士指出、普遍认为

**问题**：引用权威却不给具体来源，让AI显得在"背书"

**改写原则**：删掉归因句，或改成具体场景描述

### 6. 提纲式"挑战与未来展望"结尾
**特征词**：挑战与机遇并存、未来展望、值得深思、亟需解决、需要进一步探索

**问题**：文章结尾突然变得像学术论文大纲

**改写原则**：用具体悬念/人物下一步行动替代空泛展望

## 二、语言/语法模式（6种）

### 7. 过度使用的"AI词汇"
**特征词**：此外、然而、因此、综上所述、值得注意的是、深入探讨、至关重要、不可或缺

**问题**：这些词在AI文中出现频率远超人类写作

**改写原则**：用"不过/其实/说白了/反正"替代"然而/因此"；用"说真的/说实话"替代"值得注意的是"

### 8. 避免使用"是"
**特征**：AI 用"作为/代表/标志着/体现着"替代简单的"是"

**问题**：系动词回避是AI的典型特征

**改写原则**：该用"是"就大胆用，不要每个"是"都替换成其他词

### 9. 否定式排比
**特征词**：不是……也不是……更不是……、既不……也不……更不……、不能……也不应……

**问题**：AI 用否定排比制造"全面分析"的假象

**改写原则**：肯定句直接说，否定排比要拆散或删减

### 10. 三段式法则过度使用
**特征**：A 和 B 和 C、三大原因、三个优势、首先/其次/最后

**问题**：AI 强行把想法分成三组，显得机械

**改写原则**：写两点比写三点自然；不要每件事都凑"三"

### 11. 刻意换词（同义词循环）
**特征**：一会儿用"药物"、一会儿用"药品"、再一会儿用"药剂"；一会儿"患者"一会儿"病人"

**问题**：AI 为避免重复而过度使用同义词，反而显得刻意

**改写原则**：同一术语保持一致；重复是正常的，不要为躲重复而换词

### 12. 虚假范围
**特征词**：从 X 到 Y（从山村到城镇、从国内到国外）、涵盖 A/B/C/D

**问题**：X/Y 或 A/B/C/D 堆砌却没有实际意义

**改写原则**：删掉虚假范围句，直接说重点

## 三、风格模式（6种）

### 13. 破折号过度使用
**特征**：——这个——那个——另外——还有

**问题**：每句话都用破折号连接，显得句句都重要

**改写原则**：破折号每章不超过 3 次；能用逗号/句号解决的不要用破折号

### 14. 粗体过度使用
**特征**：大量 `**加粗**` 标记

**问题**：自媒体/网文正文不需要那么多加粗，重点用上下文体现

**改写原则**：网文正文去掉所有加粗；对话和叙述不用加粗强调

### 15. 内联标题垂直列表
**特征**：一、xxx；二、xxx；三、xxx（每个都是短标题+冒号）

**问题**：AI 喜欢用列表形式"显得有条理"

**改写原则**：把列表改写成连贯段落；标题只在大结构用

### 16. 标题中的大写
**特征**：正确使用「」却不分场合大写英文

**问题**：AI 喜欢在标题里堆砌大写英文单词

**改写原则**：网文正文不加大写英文；地名/人名用中文

### 17. 表情符号
**特征**：标题或列表前后加 🎯🔥⭐✅⚠️ 等

**问题**：网文正文不需要表情符号装饰

**改写原则**：正文去掉所有表情符号

### 18. 弯引号
**特征**：AI 倾向使用「」或『』作为对话引号

**问题**：中文网文标准对话引号是 ""（直角引号）

**改写原则**：对话统一用 ""，不用「」

## 四、交流模式/填充词（6种）

### 19. 协作交流痕迹
**特征词**：希望这对您有帮助、希望以上信息有用、如果您有任何问题、让我们一起、感谢您的阅读

**问题**：AI 结尾突然变得"服务周到"

**改写原则**：正文不留"希望有帮助"类结尾；对话里也不留

### 20. 知识截止日期免责声明
**特征词**：截至2023年/截至我的知识截止日期、我的训练数据是……、根据我截至……的了解

**问题**：学术/说明文AI结尾会加免责声明

**改写原则**：小说正文绝对不能出现此类句式

### 21. 谄媚/卑躬屈膝语气
**特征词**：当然/毫无疑问/无可否认、这确实/的确、您的观点非常正确、承蒙您指导

**问题**：AI 过度使用承认不确定性或赞美读者的句式

**改写原则**：乡土文少用"当然/毫无疑问"；多用"其实/说白了/俺寻思"

### 22. 填充短语
**特征词**：总而言之、值得注意的是、实际上、事实上、从某种意义上说、可以说、严格来说

**问题**：这些词本身没问题，但AI用得太频繁

**改写原则**：每章填充短语不超过 2 次；能用短句替代就别用短语

### 23. 过度限定
**特征词**：可能、也许、大概、应该、似乎、看上去仿佛、一般来说、基本上

**问题**：AI 不确定时用限定词回避，显得不自信

**改写原则**：有把握的事直接说；乡土文主角说话不用"可能/大概"

### 24. 通用积极结论
**特征词**：只要努力就一定能够、未来一定会更加美好、光明的前途在等待着……、最终必将

**问题**：AI 结尾强行乐观正面

**改写原则**：留悬念比给结论更抓人；人物说话符合性格即可

## 快速检查清单（写完对照）

每章写完后，逐项扫描：

```
[ ] 没有"作为/标志着/是……的体现"
[ ] 没有"专家认为/行业报告显示"
[ ] 没有句末 -ing 结构
[ ] 没有"无缝/直观/充满活力"类广告词
[ ] 没有模糊归因
[ ] 没有"挑战与机遇并存/未来展望"类结尾
[ ] 没有"此外/然而/因此/深入探讨"（每章 <= 2 次）
[ ] 没有大量否定式排比
[ ] 没有 A-B-C 三段式堆砌
[ ] 没有同一事物反复换词
[ ] 破折号每章 <= 3 次
[ ] 正文无加粗、无表情符号
[ ] 对话引号用 "" 而非「」
[ ] 没有"希望对您有帮助"类结尾
[ ] 没有"截至……"类免责声明
[ ] 没有"当然/毫无疑问"类谄媚词
[ ] 填充短语每章 <= 2 个
[ ] 不过度使用"可能/也许/大概"
[ ] 没有强行乐观结论
```
