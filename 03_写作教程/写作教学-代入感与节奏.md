# 写作教学 - 代入感与节奏控制

## 一、塑造代入感的六大核心技巧

### 1. 基础信息交代与标签化

- 读者先'认知'你的主角，再认知这个世界是什么样的世界
- 首先，你要让读者认为主角就是我，起码要让你的读者群体在阅读文字的时候，就能在脑海中将角色的外貌、衣着、性格、特征、来初步形成一个'可想象的空间'
- 这部分的篇幅不需要太多，百十来个字概括一下就行，起码让读者知道男女，高矮胖瘦，有什么特征，这就是标签化的一个过程

**总结：主角是谁、长什么样、在哪、正在经历什么，这就是基础信息的交代。**

### 2. 具体化、可视化的熟悉感

- 描述的时候要具体到某一种让读者脑海中能够不自觉浮现出来的东西
- 可视化是读者能够通过你的文字去想象，他曾经在日常生活中真实接触到的东西
- 熟悉感：这种切入的故事基础，具备天然的代入感

### 3. 共鸣｜情绪的共鸣以及认知的共鸣

- 认知共鸣：读者与书中角色'共鸣'，只有读者从内心完全认同主角
- 情绪共鸣：如何塑造一种能够在开篇就带动读者的强情绪

**示例（亲情为纽带）：**
- 捡废品抚养你长大的奶奶，被混混欺负
- 战神的女儿住狗窝
- 亲妹妹住院，急需手术费但是你打工的店长想方设法克扣原本就属于你的工资

### 4. 欲望和好奇心

- 网络小说的本质就是满足读者的'欲望'！
- '造梦'创造一个读者现实生活中'不可能去做''去拥有''不切实际'的一切
- 基础欲望：不劳而获、高人一等、权势地位
- 主动欲望（期待感）：基于你这部小说故事剧情从而让读者产生的阅读欲望和好奇心

### 5. 文笔上面的五感代入

- 五感描写：视觉、听觉、嗅觉、触觉、味觉
- 依靠读者能够具象化的描述，去依靠五感来构建环境氛围
- 示例：潮湿的短袖黏在后背上、医院消毒水的味道

### 6. 人设与代入感

- 一个好的人设甚至可以弥补剧情上的不足
- **不要是一个十全十美的人设！**
- 角色需要成长空间
- 角色的不完美，以及一些微不足道的小缺点，反而会让读者觉得更加平易近人

## 二、增强代入感的四种方式

1. **熟悉感**：之前也讲到了，熟悉的物品、口头禅、场景等等
2. **标签**：人设的标签一定要提前构思好，以及他的小缺点
3. **冲突的紧迫性**：通过危机的紧迫性，更能让读者快速沉浸下去
4. **接地气**：依靠人物缺点，又或是很多白话不高深不费脑的描述

## 三、常见写作雷区

### 1. 写作人称

- 第一人称：以"我"为主体
- 第二人称：基本没人用
- 第三人称：最常见，上帝视角，自由切换

**新手建议：** 第三人称是最顺手的，也是比较好去做写作练习的

### 2. 避免流水账

**两个办法：**
1. **增加冲突**：给你的流水账，增添前因后果，也就是增添故事冲突
2. **加入强情绪**：让你的剧情能够让读者跟着你一起着急

### 3. 开篇不吸引人

- 开篇从 `爆点` 入，不要从"砌砖"开始
- 前三章必须瘦，不能肥
- 默认用 `黄金三章`
  1. 抛出主冲突
  2. 亮出主角的特殊优势或系统
  3. 锁定短期目标

### 4. 人设有问题

**常见错误：**
- 所有故事角色说话的语气都是同一个语气
- 所有角色发脾气的时候，都是同一种表现模式
- 故事角色的前后人设不统一

**公式：核心标签 + 反差细节 = 活人**

### 5. 其他问题

- 不要故意秀文笔！不要做词藻堆砌！
- 钩子问题：每一章的结尾，都给读者留好钩子
- 书名和简介：书名尽量做到：`题材 + 核心爽点 + 主角行为/身份`
