# Quick Reference - 快速查阅卡

写90年代小说时，先看这个！

⚠️ **核心原则：同一本书内数据保持一致即可！不要纠结精确数字！**

---

## 🔥 绝对不能写错的核心时间节点

| 时间 | 事件 | 写作注意 |
|------|------|---------|
| **1993年** | 全国全面取消粮票、布票等所有票证 | 1993年之前必须凭票买米面油，不能写随便用现金买 |
| **1994年** | 分税制改革，当年通胀率24.1% | 物价大幅上涨，是剧情冲突的好节点 |
| **1997年** | 亚洲金融危机，国企下岗潮全面爆发 | 中专取消包分配，热度快速下降 |
| **1998年** | 房改全面启动，停止福利分房 | 商品房时代开启 |
| **2000年之前** | 农村必须缴纳农业税和提留统筹 | 直到2006年才全面取消农业税 |

## ⚠️ 绝对不要犯的错误

| 常见错误 | 正确写法 |
|----------|---------|
| 90年代一碗面10块钱 | 90年代一碗面0.5-1元 |
| 90年代一斤肉20块钱 | 90年代一斤肉1.8-2.2元（1990年）或6-7元（1995年） |
| 主角一个月赚10万 | 主角一年赚1000-3000元就是首富 |
| 底层农户年入几万 | 底层农户年现金收入200-400元 |
| 1990年随便用现金买米面 | 1993年之前必须凭粮票 |
| 1990年全村家家有电视 | 1990年农村通电率约80%，电视拥有率更低 |
| 1990年农村到处是汽车 | 主要运输工具是拖拉机、牛车、自行车 |

## 📌 货币&计量常识

- **90年代通用第四套人民币**，1999年10月1日才发行第五套人民币
- 1999年之前不能出现100元新版人民币
- 农村日常计量通用**市斤（斤）、亩、尺**，极少用公斤、公顷、米
- 口语里钱的单位是"**块、毛**"，极少用"元、角"

## 🌾 乡土文高频细节速查

### 赶集
- 农村逢集一般是**农历逢一、四、七**或**二、五、八**
- 赶集是农村最主要的消费和社交场景
- 主要交易：农产品、日用百货、服装、牲畜

### 交通
- 90年代农村主要运输工具是**拖拉机、牛车、自行车**
- 极少有汽车
- 外出打工多坐**绿皮火车**或**长途客车**

### 通电
- 1990年农村通电率约80%
- 很多偏远村子1993-1995年才通稳定的电
- 1990年不能写全村家家有电视

## 📌 关键原则

1. **同一本书内物价保持一致！**
2. **选择一个合理范围，不要纠结精确数字！**
3. **写北方农村参考黑龙江数据，写南方可以高10-20%！**
4. **官方收入数据包含实物（自产粮食），现金收入更低！**
5. **底层农户一年到手现金可能不到300元！**
6. **1993年之前买米面油必须凭票证！**
7. **农村日常用市斤、亩、尺，口语说块、毛！**

## 🔍 获取真实数据的方法

### 1. 官方来源（最权威）
- **国家统计局官网**：www.stats.gov.cn
  - 搜索"1990年统计公报"
  - 搜索"农村居民贫困状况"

- **地方志/史志网**
  - 黑龙江史志网、天津物价志等
  - 找"主要农产品价格"、"物价"等章节

### 2. 对比多个来源
- 官方数据 vs 地方志 vs 网友回忆
- 不同地区的数据有差异（江苏 > 山东 > 黑龙江）
- 选择一个合理范围，同一本书内保持一致
