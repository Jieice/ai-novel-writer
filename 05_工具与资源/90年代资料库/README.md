## 🔥 绝对不能写错的核心时间节点

| 时间 | 事件 |
|------|------|
| **1993年** | 全国全面取消粮票、布票等所有票证 |
| **1994年** | 分税制改革，当年通胀率24.1% |
| **1997年** | 亚洲金融危机，国企下岗潮全面爆发，中专取消包分配 |
| **1998年** | 房改全面启动，停止福利分房 |
| **2006年** | 全面取消农业税 |

## 💡 使用建议

1. **写小说前先看 QuickReference**，掌握核心时间节点和避坑指南
2. **物价和收入不要穿越**：90年代一碗面0.5-1元，一斤肉1.8-2.2元（1990年）
3. **赚钱速度要合理**：半年赚500-1000元全村震惊，一年赚1000-3000元村里首富
4. **同一本书内数据保持一致**，不要纠结精确数字
5. **写北方农村参考黑龙江数据，写南方可以高10-20%**
6. **1993年之前买米面油必须凭粮票**，不能写随便用现金买

## 📌 货币&计量常识

- **90年代通用第四套人民币**，1999年10月1日才发行第五套人民币
- 农村日常计量通用**市斤（斤）、亩、尺**，极少用公斤、公顷、米
- 口语里钱的单位是"**块、毛**"，极少用"元、角"

## 🌾 乡土文高频场景提示

- **赶集**：农历逢一、四、七或二、五、八，农村最主要消费和社交场景
- **盖房**：1990年盖三间砖瓦房3000-5000元，1999年15000-20000元
- **学费**：1990年小学一学期不超15元，1999年不超110元
- **农业税**：1990年亩均20-30元，1999年亩均80-120元，剧情冲突来源
